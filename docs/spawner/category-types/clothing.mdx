id: clothing
title: Clothing
---

Example Configuration:

```lua
{
	title = "My Category",
    // highlight-next-line
	categoryType = "clothing",
    // highlight-next-line
    assetId = 12345678,
}
```

To set a category to a clothing item - set the `categoryType` to `"clothing"`

Then, set your `assetId` to the asset Id of the clothing article

:::note

Type clothing makes it so anyone with a certain clothing item owned can spawn a bus

:::
